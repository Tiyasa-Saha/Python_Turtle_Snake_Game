# Python_Turtle_Snake_Game
The snake game you can play just like you did in the Nokia phones once upon a time.

The code has 4 files:
1. main.py
2. food.py
3. snake.py
4. scoreboard.py

Run the main.py file in any code editor to play the game. It works just like the regular Nokia snake game:
1. Game over when you hit a wall of the screen.
2. Game over if you hit the snake body.
3. Score a point on eating the food.
4. Game speed increases every 5 points.
5. The snake grows longer with every point gained/every time it eats.
